# Mechanical Assembly — SolidWorks

This project contains a SolidWorks mechanical assembly with individual part drawings, an exploded view, and a final assembly drawing.

## Project Overview

The assembly consists of a top plate, axle supports, axle, bushings, wheel, bolts, and nuts. The project demonstrates part modeling, assembly relationships, technical drawing creation, exploded view generation, and bill of materials documentation.

## Components

- Top plate
- Axle support
- Axle
- Bushing
- Wheel
- M6 bolts
- M6 nuts

## Skills Demonstrated

- SolidWorks part modeling
- Mechanical assembly design
- Exploded view generation
- Engineering drawing layout
- Bill of materials creation
- Dimensioning and tolerancing
- Technical documentation
