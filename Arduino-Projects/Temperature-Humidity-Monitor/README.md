# Arduino Temperature and Humidity Monitor

This Arduino project monitors temperature and humidity, displays live readings, and uses warning indicators to show environmental conditions.

## Project Overview

The system reads temperature and humidity values using a sensor, displays the values on an LCD screen, and uses LEDs and/or a buzzer to indicate warning levels based on preset thresholds.

## Components

- Arduino board
- Temperature and humidity sensor
- LCD display
- LEDs
- Buzzer
- Breadboard and jumper wires

## Features

- Reads temperature and humidity
- Displays live values
- Uses warning indicators for different environmental conditions
- Applies conditional logic based on sensor readings

## Skills Demonstrated

- Arduino programming
- Sensor input
- LCD display control
- Conditional logic
- Embedded systems basics
- Hardware/software integration
