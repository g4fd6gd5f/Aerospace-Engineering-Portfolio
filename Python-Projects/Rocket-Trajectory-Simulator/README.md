# Rocket Trajectory Simulator with Drag

This project is a Python simulation of two-dimensional projectile motion with aerodynamic drag.

The program calculates the flight path of a projectile or simplified rocket using initial velocity, launch angle, mass, drag coefficient, cross-sectional area, air density, and time step. It outputs flight summary data, saves simulation results to a CSV file, and generates trajectory and velocity plots.

## Project Overview

The goal of this project was to apply Python programming to an aerospace-related simulation problem. The simulation models projectile motion while accounting for drag force, making it more realistic than ideal projectile motion without air resistance.

The program calculates:

- Horizontal position over time
- Altitude over time
- Velocity components
- Total speed
- Drag force
- Flight time
- Range
- Maximum height

## Inputs

The user enters:

- Initial velocity in m/s
- Launch angle in degrees
- Mass in kg
- Drag coefficient
- Cross-sectional area in m²
- Air density in kg/m³
- Time step in seconds
