# C++ Projects

This folder contains selected C++ projects demonstrating programming fundamentals, algorithmic problem-solving, and engineering-based calculations.

## Projects

### Beam Stress and Deflection Calculator

A C++ engineering calculator for basic beam analysis. The program allows the user to select different beam support and loading cases, then calculates structural outputs such as reactions, maximum shear force, maximum bending moment, bending stress, deflection, and factor of safety.

**Concepts demonstrated:**
- Functions
- User input validation
- Conditional logic
- Mathematical modeling
- Engineering calculations
- Mechanics of materials
- Structural analysis fundamentals

**Engineering topics:**
- Simply supported beams
- Cantilever beams
- Point loads
- Uniformly distributed loads
- Bending stress
- Beam deflection
- Factor of safety

**File:** [`StressBeamCalculator.cpp`](./StressBeamCalculator.cpp)

---

### Sudoku Validator

A C++ program that checks whether a 9x9 Sudoku grid satisfies row, column, and 3x3 region requirements.

The program uses separate validation functions for:
- Rows
- Columns
- 3x3 regions

**Concepts demonstrated:**
- 2D arrays
- Functions
- Pointers
- Nested loops
- Boolean logic
- Algorithmic validation

**File:** [`sudoku_validator.cpp`](./sudoku_validator.cpp)

---

### Pattern Box Generator

A console-based C++ program that asks the user for a number of rows and columns, then prints a bordered text box using letters.

The program limits the number of rows to 26 so the output remains within lowercase alphabet characters.

**Concepts demonstrated:**
- User input
- Nested loops
- Conditional statements
- Character manipulation
- Console output formatting

**File:** [`pattern_box_generator.cpp`](./pattern_box_generator.cpp)
